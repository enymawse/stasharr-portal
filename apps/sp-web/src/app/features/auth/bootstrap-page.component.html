First-run protection

Create the Stasharr admin account

Before anyone on your LAN can use Stasharr, create the single local admin account that protects this instance.

This account signs into the app, unlocks setup, and can change its password later.

After this, Stasharr opens required integration setup for catalog, Stash, and Whisparr.

Use a strong password now even if this is a private home network install.
